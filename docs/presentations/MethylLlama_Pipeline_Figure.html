Figure. Overview of the MethylLlama pretraining and fine-tuning pipeline. (1) Raw DNA methylation profiles from Illumina arrays (β ∈ [0,1], ~850K CpG sites, mixed tissues). (2) Dual-field tokenization: each CpG is represented as a (ID, β-value) pair; missing values are excluded. (3) WCED (Weighted CpG Encoder Decoder) self-supervised pretraining: a random 50% subset of 8,000 CpGs is used as input; the model reconstructs held-out CpGs via the CLS embedding, supervised additionally by an age auxiliary loss and an InfoNCE contrastive loss on two random views. (4) MethylLlama architecture: LLaMA-style encoder with 256D hidden size, 4 layers, 4 attention heads, RMSNorm Pre-LN, RoPE, and SwiGLU FFN (~7.7M parameters). (5) Fine-tuning for biological age prediction: the pretrained encoder (frozen for 10 epochs) feeds the CLS pooler output into a 3-layer MLP head. (6) Downstream results: V5 achieves test MedAE = 3.65 yr, R² = 0.905 on the AltumAge benchmark using only 19,608 CpGs.
