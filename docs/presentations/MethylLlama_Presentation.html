The Problem — What Are We Trying to Learn?

DNA Methylation (β values)

Each CpG site has a methylation level β ∈ [0, 1].
Low β = unmethylated. High β = methylated.

These patterns encode tissue identity, biological age, and disease state.

The Scale Challenge

  • 450k / 850k CpG arrays → only 49,156 high-quality type-III probes used
  • Each sample = a vector of 49k real numbers in [0, 1]
  • 169,120 samples × 49,156 CpGs = ~21 GB dense matrix

Goal: Learn a Universal Methylation Encoder

  • Pretrain on 170k samples — no labels needed
  • Encode full methylome → single CLS vector
  • Fine-tune on downstream tasks (age prediction, tissue classification)
Key insight: If we can predict 50% of CpGs from the other 50%, the encoder must have learned meaningful biological structure — not just memorised noise.

ScaleAdapt — How Beta Values Are Encoded

Formula

For each β ∈ [0, 1] and k = 1..48 basis functions:

features_k = [sin(β × f_k), cos(β × f_k)]
output = Linear(96 → 256)(concat all features)

Frequencies f_k are initialised from N(0, 2π × 2.0) and learned during pretraining — the model decides which resolution of β variation matters biologically.

Special Token Routing

  • β = -1 → MASK token embedding
  • β = -2 → CLS token embedding
  • β = -3 → SEP token embedding
  • β = -4 → PAD token embedding
  • β ≥ 0 → sinusoidal encoder (real methylation value)

Why Not MLP?

  • MLP: 0.85 and 0.87 produce similar hidden states — small difference is lost
  • ScaleAdapt: high-frequency basis creates large angular separation for small β differences
  • Same principle as positional encoding in NLP — applied to continuous values

basis_scale = 2.0

Upstream model used 1.5 (scRNA, range 0–10+).

Methylation β ∈ [0,1] is a much tighter range → need higher initial frequencies to resolve fine differences.

Value of 2.0 tuned for the methylation domain.

Loss Function

Only on held-out CpGs · NaN positions excluded

Why This Works

  • Forces encoder to learn co-methylation patterns across the genome
  • CpGs in the same region are correlated — encoder must learn this structure
  • No labels needed — fully self-supervised on 170k samples
  • CLS vector must summarise the full methylome to reconstruct it

Optional Extensions (currently disabled)

  • Contrastive (InfoNCE): Two random 50% views of same sample = positive pair. Improves CLS representation quality
  • Age supervision: AgeHead on CLS forces biological age into the representation

The Hypothesis

Methylation data is highly structured — CpGs in the same genomic region are strongly correlated. A small model with enough data may capture this structure as well as a large model.

Data > Parameters for Methylation

  • 170k samples cover broad tissue and age diversity
  • 49k CpGs = rich input signal per sample
  • MethylGPT (64D, ~6M) achieved strong results on similar data
  • Our small model adds RoPE, SwiGLU, ScaleAdapt on top

Practical Benefit

  • 4 GPUs → 300 epochs in ~24h (vs ~5 days for full model)
  • Faster iteration cycle → more experiments possible
  • If small ≈ full: lighter deployment, lower inference cost

Pretraining Data — 169k Methylomes, 49k CpG Probes

Dataset Overview

  • 169,120 samples × 49,156 CpG probes (type-III Illumina EPIC array)
  • Stored as 33 GB h5ad — loaded lazily row-by-row (never fully in RAM)
  • Source: GEO public methylation studies (TWPID cohort)
  • No age / tissue labels — fully self-supervised
  • Split: random 80 / 10 / 10 (no pre-existing split column)

What is a Beta Value?

β = (methylated signal) / (methylated + unmethylated signal)

β ∈ [0, 1] — 0 = fully unmethylated, 1 = fully methylated

Type-III probes: consistent measurement across tissue types. Used for cross-tissue generalisation.

Why ~8.5% NaN in Pretrain?

  • Some Illumina probes fail per sample (low intensity, SNP overlap, cross-hybridisation)
  • NaN = "probe not measured" — biologically meaningful absence
  • WCED collator excludes NaN positions from loss — no imputation needed
  • Model learns to reconstruct from partial observations → naturally robust to missing probes

Why 60% NaN — and Why It's Fine

  • Fine-tuning samples come from multiple array platforms — each measures only a subset of the 49k type-III probes
  • NaNs are scattered (not a contiguous block) — random per sample
  • On average, ~19,660 valid CpGs per sample — enough signal for age prediction
  • Pretrain was trained with ~50% masking (24,578 visible CpGs) — the model already learned from partial observations
  • WCED collator excludes NaN from loss — no change needed to handle 60% NaN
Key insight: 60% NaN in fine-tuning ≈ 50% masking in pretraining. The encoder was explicitly trained for this scenario — predicting from incomplete observations is the pretraining task itself.

Architecture Decision — Which Model to Proceed With?

Why h256_l4 is the Primary Candidate

  • Nearly identical val PCC to h512_l2 (0.884 vs 0.886 — within noise)
  • 2× fewer parameters (16.8M vs 33.6M) → faster training, cheaper inference
  • Already running on full 170k data — epoch 24: val/pcc = 0.962
  • Proven stable: cpg_scale reached 1.98 (up from 0.1 init)
  • grad_norm = 0.006 — extremely stable training

Why Not h512_l2 (sweep winner)?

  • Only +0.002 PCC over h256_l4 — within noise on 2k samples
  • 2× more parameters with no meaningful gain
  • Sweep on 2k samples cannot reliably distinguish architectures
  • Efficiency favours h256_l4 for rapid iteration on full data

Smoke Test Validated the Pipeline

  • Smoke test (2k samples, 30 epochs): val PCC = 0.887
  • Confirmed: data loading, tokenizer, WCED collator, WandB all work end-to-end
  • Architecture differences on 2k samples are within noise — full data needed for real comparison
  • h256_l4 selected as the best efficiency/performance trade-off to launch full pretrain

Next Steps

  • Continue full pretrain to convergence (~200 epochs)
  • Linear probe: freeze encoder → predict age with linear layer
  • Full fine-tuning: age prediction and tissue classification
  • If h256_l4 plateaus: launch h512_l2 on full data for comparison

Summary

Data Pipeline

  • 169k GEO samples · 49k type-III CpG probes
  • Stored as 21 GB h5ad, loaded lazily row-by-row
  • Random 50/50 split per epoch (WCED collator)
  • 8.5% NaN rate handled by loss masking

Architecture (Small Model)

  • 4 layers × 256D × 4 heads · SwiGLU FFN
  • RMSNorm Pre-LN · RoPE (no position table)
  • ScaleAdapt sinusoidal β encoder (48 basis, scale=2.0)
  • Learned cpg_scale parameter (init=0.1)
  • WCEDDecoder: CLS → Sigmoid → 49k β̂

Key Changes vs SCBert

  • Pre-LN + RMSNorm → stable training from step 1
  • SwiGLU → more expressive FFN
  • RoPE → no position embedding table
  • ScaleAdapt → better β value resolution
  • No monkey-patching → clean, maintainable codebase

Core Hypothesis

  • Methylation is highly structured → small model may suffice
  • Full data (170k) matters more than model size
  • Modern LLaMA components improve over BERT baseline
  • Self-supervised WCED → universal methylome encoder
