Design Choices vs Standard BERT

RoPE
Relative position encoding inside attention — no fixed position table. The model understands which CpG is where without memorizing positions.
RMSNorm
40% cheaper than LayerNorm, applied before each sublayer (Pre-LN) for stable training from step 1.
SwiGLU
Gated MLP activation (same as LLaMA, GPT-4). More expressive than standard GELU — better at learning complex methylation patterns.
ScaleAdapt
Trainable sinusoidal encoder for beta values [0,1]. The methylation value (0 = unmethylated, 1 = methylated) is encoded as a continuous signal, not discretized.

Why Random 50% Every Batch?

If the same CpGs were always input and the same CpGs always target, the model would memorize fixed input→output mappings — no real generalization.

By randomly shuffling which 50% are input vs target at each batch, the model is forced to learn global methylation patterns. It can never rely on specific CpGs always being available.

At inference: feed any subset of CpGs → robust representation. This is the same principle as MAE (Masked Autoencoders) in vision.

Why Contrastive Loss?

Without contrastive loss, the CLS representation collapses — the encoder produces nearly identical vectors for all samples. The reconstruction task alone does not require diverse representations.

InfoNCE: two different random 50% views of the same sample should produce similar CLS embeddings; different samples should be pushed apart.

Total loss = Reconstruction MSE + 0.1 × InfoNCE

This ensures the CLS embedding is discriminative enough for downstream tasks.
