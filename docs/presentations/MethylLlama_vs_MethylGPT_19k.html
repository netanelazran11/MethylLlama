MethylLlama's entire fine-tuning fits inside the first 2.3% of MethylGPT's training
Why the 43× step gap?
MethylGPT uses batch_size = 1 — each gradient step updates weights on a single sample. MethylLlama uses batch_size = 64, so each step aggregates gradients over 64 samples. This explains the large step gap even though MethylLlama trained for more epochs (300 vs 99). WCED pretraining additionally gives MethylLlama a strong initialisation — the model converges to R² ≥ 0.90 within ~5,000 steps, whereas MethylGPT requires ~430,000 steps for the same quality.
