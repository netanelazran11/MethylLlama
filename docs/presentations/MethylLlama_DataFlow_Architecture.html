CpG VALUE TYPES
0.0
Real data — Unmethylated CpG
beta = 0.0 is a valid measurement.
It means the CpG site is not methylated in this sample.
Routed through ScaleAdaptEncoder (sinusoidal encoding).
NaN
Structural missing — Array not measured
29,548 CpGs (of 49k vocab) are never measured on 450k arrays.
Stored as NaN in h5ad — never tokenized, never fed to encoder.
WCEDCollator builds input only from non-NaN positions.
−2
Special token — CLS marker
CLS token has beta = −2.0 (negative = special).
Routed through learned nn.Embedding (not ScaleAdapt).
Placed at position 0 of every sequence.
