01 Overview & Research Goal

DNA methylation patterns change systematically with age, making them powerful biomarkers for biological age prediction. We start from MethylGPT (a GPT-style foundation model for methylation) as our baseline, then propose adapting BMFM-RNA — an IBM research foundation model originally built for single-cell RNA-seq — to the methylation domain. We explore two distinct pretraining strategies (MLM and WCED) and compare them against MethylGPT on a held-out test set.

Approach 1 — MethylGPT

GPT-style Transformer (scGPT backbone). Single-field tokenization. MLM pretraining on 226k profiles. ResNet1D fine-tuning head.

Approach 2 — BMFM-RNA (MLM)

SCBert backbone adapted for methylation. Dual-field tokenization (site ID + beta-value). MLM pretraining, then MLP age head. Mean pooling.

Approach 3 — BMFM-RNA (WCED)

Same SCBert backbone. Contrastive Cross-view Encoder Distillation pretraining. CLS token explicitly trained to encode full methylation profile. Three fine-tuning strategies.

Core Hypothesis
BMFM-RNA's dual-field tokenization — which separately encodes CpG site identity and methylation value — should better preserve fine-grained methylation information than MethylGPT's implicit single-field approach. Additionally, the WCED pretraining forces the encoder's CLS token to serve as a rich profile-level summary, which can then be directly frozen for downstream regression without further encoder updates.

WCED Pretraining Phase

Instead of simple masking, WCED creates two random subsets of CpG sites (views) from each sample. Both views pass through the same encoder. The CLS token is then used to (1) reconstruct the full profile, (2) match the two views via contrastive loss, and (3) predict age as an auxiliary signal. This forces CLS to encode the complete methylation profile.

05 Tokenization & Input Representation
Why Tokenization Matters
Each CpG site has two pieces of information: which site it is (genomic context, chromosome location) and what value it has (β ∈ [0,1], the methylation level). How you encode these two aspects fundamentally affects what the model can learn.
MethylGPT — Single-Field Tokenization
token_i = Embed(CpG_ID) + ValueEncoder(β_i)

Both site identity and beta value are combined into a single embedding space. The model cannot independently adjust how much weight to give to site context vs. value signal. Vocabulary: CpG site tokens only.

Vocabulary size~8,005 tokens
Encoding schemeImplicit combination
Value encoderEmbedded with CpG ID
BMFM-RNA — Dual-Field Tokenization
h_i = e_cpg(s_i) + e_β(β_i)

Two separate encoders process site identity and value independently, then combine additively. This allows the model to learn which sites are informative independently of their current value. β-values are encoded as continuous MLP projections (no discretization).

How Masking Works in BMFM-RNA (MLM Pretraining)
🎯
Step 1: Select Candidates
25% of all tokens randomly selected as masking candidates (~2,000 of 8,000 CpGs)
🎭
Step 2: Apply Masking
30% of candidates → [MASK] token in Field 2. Others retained as-is (helps prevent bias).
🔮
Step 3: Reconstruct
Model predicts the original β-values at masked positions. Loss = MSE on masked sites only.

10 Conclusions & Key Insights
1. MethylGPT Wins Overall (Actual Results)
With only ~2M parameters, MethylGPT achieves the best test MAE (4.45 yrs) and test R² (0.9394) across all methods. BMFM approaches with 10–13× more parameters do not surpass it. This reveals a strong performance ceiling in the task itself, not model capacity.
2. WCED is Best BMFM Strategy
Among all BMFM variants, WCED Method 1 (frozen encoder + fixed 8k CpGs) achieves the best result (MAE=4.61, R²=0.9327), closely competitive with MethylGPT. The contrastive + reconstruction + age pretraining signal produces a significantly richer CLS representation than MLM alone.
3. Don't Modify the WCED Encoder
For WCED, any encoder fine-tuning degrades performance: Frozen (4.61) > Decoder FT (5.22) > LoRA (6.83). The pretrained CLS embedding is already optimal — any interference destroys the learned representation. In contrast, BMFM-MLM benefits from careful encoder unfreezing at epoch 5.
4. Capacity ≠ Performance Here
MethylGPT (~2M params) outperforms BMFM MLM (~23M params) by 0.39 yrs MAE and 1.4% R², despite 11× fewer parameters. WCED (~27M params) is only marginally better than MethylGPT (+0.16 yrs MAE). The task ceiling limits further gains regardless of capacity.
5. Pooling Strategy is Architecture-Dependent
After MLM pretraining, CLS receives no aggregation training signal → mean pooling over content tokens is correct. After WCED pretraining, CLS is explicitly trained as a full-profile summary → pooler_output (CLS only) is correct. Wrong pooling choice degrades performance significantly.
6. Multi-Seed Evaluation is Essential
WCED results are from a single run — fair comparison requires multi-seed evaluation. MethylGPT's 4-seed std is very low (R²: ±0.001), confirming stable training. BMFM MLM shows higher variance (R²: ±0.005). Seed 42 was excluded from MethylGPT due to suspected val/test split artifact (best test but worst val).
Paths to Better Results
1. Multi-Seed WCED Evaluation
WCED M1 is currently a single run (MAE=4.61). Running 5 seeds would either confirm it matches MethylGPT or reveal variance. This is the most direct next step — no code change required, just submit more jobs.
2. Clean Data — Filter Placenta & Sperm
The 8k dataset contains ~517 non-aging samples (placenta, sperm) with age=0 that add wrong signal. The 21k dataset has tissue labels enabling precise removal. Training on clean data is expected to lower MAE by removing noise from the age distribution boundary.
3. WCED Pretraining with Full 21k CpGs
Pretraining on 21k CpGs covers 2.6× more methylation sites, giving the encoder a broader view of the epigenome. Fine-tuning with all 21k as input (vs. 8k) captures more age-informative CpG sites and may push R² above 0.94.
4. Larger / Deeper Encoder
Current BMFM encoder: 6 layers, 512d, ~23M params. Scaling to 12 layers / 768d (~85M) with Flash Attention on H200 is feasible. Given that WCED M1 already rivals MethylGPT at 27M params, a larger encoder with the same WCED objective may be the clearest path to surpassing the current ceiling.
Bottom Line
MethylGPT is the best overall method — Test R²=0.9394, MAE=4.45 yrs (4 seeds), with only ~2M parameters.
WCED M1 is the best BMFM variant — Test R²=0.9327, MAE=4.61 yrs — close to MethylGPT but with 13× more parameters and a single run.
BMFM MLM ranks third — Test R²=0.927, MAE=4.79 yrs (5 seeds) — stable across seeds, but dual-field tokenization alone does not surpass MethylGPT.
The key finding: pretraining objective and tokenization design matter more than model capacity. WCED's contrastive + reconstruction + age signal produces richer representations than MLM alone, but a full multi-seed WCED evaluation is needed to confirm the gap vs. MethylGPT.
11 Pretraining Data Effect & Future Directions
Central Question: Is the gap between MethylGPT (MAE=4.45) and BMFM WCED M1 (MAE=4.61) due to a weaker model — or simply because BMFM was pretrained on the same small dataset it was fine-tuned on? The evidence points strongly to the pretraining data as the dominant factor.

These are fundamentally incomparable conditions. The 0.16 MAE gap likely reflects pretraining data scale, not model architecture.
How Much Does Pretraining Data Scale Matter?
In NLP (BERT family)
Scaling pretraining data from 300M → 3B tokens: ~1–3% downstream improvement across GLUE tasks. But methylation has far fewer samples available than text — so each extra sample has higher marginal value. With only 9k pretraining samples, the model is almost certainly in the data-starved regime.
In Methylation (estimated effect)
Going from 9k → 100k pretraining samples on methylation data could realistically shift MAE by 0.3–1.5 years based on the observed NLP scaling trends and the high information density of methylation profiles. This alone would explain the 0.16 MAE gap — and potentially close it entirely.
Why WCED pretraining on the same 8k dataset still helps (vs. no pretraining):
Even self-supervised pretraining on 9k samples is useful — it teaches the encoder the structure of methylation co-variation, forces CLS to aggregate full-profile information (reconstruction objective), and bakes in age signal (auxiliary age loss). The question is whether a larger, more diverse pretraining corpus would produce a substantially better initialization. The answer is almost certainly yes.
Future Directions — Pretraining Data
1. Pretrain BMFM on a Large External Corpus
Collect publicly available methylation array data (ENCODE, GEO, TCGA, Blueprint Epigenome — 850k Illumina arrays). Pretrain BMFM WCED on 50k–200k samples across diverse tissues before fine-tuning on AltumAge 8k. This is the most direct way to create a fair comparison with MethylGPT's pretrained weights. Expected effect: reduce MAE by 0.2–0.8 years; close the gap with MethylGPT.
2. Use the 21k AltumAge Dataset for Pretraining
The 21k h5ad dataset contains 2.6× more CpG sites (21,368 CpGs) and more samples. Pretraining WCED on the full 21k (after filtering placenta & sperm via filter_21k_h5ad.py) provides more coverage of the methylation landscape and more training signal. Fine-tuning on the 8k subset can then be viewed as a compressed downstream task — the encoder has already seen the broader epigenomic context.
3. Cross-Tissue Pretraining → Blood Fine-tuning
Pretrain BMFM WCED on a multi-tissue corpus (blood + brain + liver + muscle methylation), then fine-tune exclusively on blood AltumAge. Multi-tissue pretraining forces the encoder to learn tissue-invariant methylation structure — the CpG co-variation patterns that are conserved across tissues are likely the most biologically fundamental. This mirrors the strategy that made MethylGPT's pretrained weights powerful.
4. Ablation: Same Data, WCED vs. MLM
The fairest comparison of pretraining objectives: train both MLM and WCED on the identical pretraining corpus (same data, same split, same CpGs), then fine-tune both under identical conditions. This isolates the objective effect from the data effect. Current results (WCED M1 > MLM by 0.16 MAE) already suggest WCED is better, but the data confound prevents a clean conclusion.
Most Practical Immediate Next Step
1. Run multi-seed WCED fine-tuning (5 seeds) — zero code change, quantify variance first.
2. Pretrain BMFM WCED on the 21k filtered dataset — larger data, more CpGs, clean labels (tissue filter already written).
3. Fine-tune the 21k-pretrained checkpoint on 8k AltumAge — compare with current 8k-pretrained WCED M1.
If step 3 shows meaningful improvement (expected: 0.2–0.5 MAE reduction), this confirms pretraining data scale is the dominant factor. If the gap to MethylGPT remains, then the focus should shift to architecture (larger encoder) or acquiring the same external corpus MethylGPT was pretrained on.
