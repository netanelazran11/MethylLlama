Adapting BMFM-RNA for DNA Methylation

Multi-Field Tokenization for Biological Age Prediction

This presentation describes how we adapted a transformer foundation model originally designed for single-cell RNA sequencing to predict chronological age from DNA methylation patterns. The key insight is that both data types share a common structure: discrete identifiers paired with continuous measurements.

Why BMFM-RNA for Methylation?

DNA methylation and gene expression data share a fundamental structure that makes BMFM-RNA's multi-field architecture directly applicable to methylation analysis.

Single-Cell RNA (Original)

  • Identifiers: Gene symbols (BRCA1, TP53, EGFR...)
  • Values: Expression counts (integers, 0 to 10,000+)
  • Features: ~20,000 genes per cell
  • Task: Cell type classification

DNA Methylation (Adapted)

  • Identifiers: CpG probe IDs (cg00000029, cg00000108...)
  • Values: Beta-values (continuous, 0.0 to 1.0)
  • Features: 8,000 CpG sites (Altumage selection)
  • Task: Age regression

Shared Data Structure

Both modalities pair discrete tokens (gene/CpG IDs) with continuous measurements (expression/methylation levels). BMFM handles this natively with its two-field design.

Transfer Learning Opportunity

The architectural patterns learned for gene expression relationships can transfer to methylation patterns, especially the attention mechanisms that capture feature interactions.

Two-Field Tokenization

The key adaptation is how we handle the two types of information in methylation data: the CpG site identifiers (which site are we measuring) and the beta-values (what is the methylation level).

Critical Design Decisions

CpG IDs Are Never Masked

During MLM pretraining, we always know which CpG site we are measuring. The site identity is structural information, not something to predict. Only beta-values are masked and reconstructed.

All 8,000 CpG Sites Used

We use all 8,000 CpG sites from the Altumage dataset for every sample. This ensures the model learns comprehensive methylation patterns across the full feature set.

Continuous Value Encoder

Beta-values (0.0-1.0) cannot use standard vocabulary lookup. Instead, a small MLP (1→128→512) projects each scalar value into the embedding space, producing a 512-dim vector per CpG site.

Embedding Fusion by Addition

The CpG ID embedding and beta-value embedding are combined by element-wise addition. This allows the model to jointly reason about which site and what value, creating position-aware value representations.

Model Architecture

We use a smaller version of the BMFM architecture, reducing capacity to prevent overfitting on the limited methylation dataset while maintaining the core transformer structure.

Component BMFM-RNA (Original) BMFM-Methylation (Ours) Rationale
Transformer Layers 12 6 Methylation patterns are more regular, need less depth
Attention Heads 12 8 Balanced for 512 hidden dim (64 per head)
Hidden Dimension 768 512 Sufficient representation capacity for methylation
FFN Intermediate Size 3,072 2,048 Standard 4x expansion of hidden dimension (512 × 4)
Vocabulary Size ~60,000 genes 8,005 8,000 CpG sites + 5 special tokens
Input Sequence Length Variable 8,000 All CpG sites from Altumage dataset
Total Parameters ~110M ~25M 4× smaller to prevent memorization

FFN Intermediate Size (2,048): This is the width of the feed-forward network inside each transformer layer. Each transformer block has: Attention → FFN. The FFN expands the 512-dim hidden representation to 2,048-dim (4× expansion), applies GELU activation, then projects back to 512-dim. This is the standard transformer design used in BERT.

Training Pipeline

Training consists of two stages: self-supervised pretraining to learn methylation patterns, followed by supervised fine-tuning for age prediction.

Stage 1: Pretraining (Masked Language Modeling)

1
Input Preparation

Load methylation profiles from h5ad file. Each sample contains beta-values for all 8,000 CpG sites.

Data: 11,500 samples total (train: 5,483, valid: 1,371, test: 4,646)
2
Full Feature Input

Use all 8,000 CpG sites for every sample to capture complete methylation profiles.

Consistent feature set enables learning global methylation patterns
3
Masking (30% of beta-values)

Randomly replace 30% of beta-values with mask token (-1.0). CpG IDs remain visible.

Higher mask rate than BERT (15%) encourages learning richer representations
4
Reconstruction Objective

Model predicts original beta-values at masked positions. Loss: MSE on masked positions only.

Teaches model to understand methylation co-regulation patterns
5
Training Configuration

AdamW optimizer, cosine LR schedule with warmup, early stopping on validation loss.

Epochs: up to 300, Batch: 32 × 4 accumulation = 128 effective

Stage 2: Fine-tuning (Age Regression)

1
Load Pretrained Encoder

Initialize from best pretraining checkpoint. Freeze all encoder parameters.

Freezing prevents catastrophic forgetting and overfitting
2
Add Regression Head

Attach trainable MLP (512→256→128→1) on [CLS] pooler output.

Only ~200K trainable parameters in the head
3
Age Normalization

Normalize target ages: z = (age - μ) / σ where μ=42.0, σ=26.53 years.

Improves training stability for regression
4
MSE Loss on Normalized Ages

Minimize mean squared error between predicted and true normalized ages.

Denormalize predictions for final MAE/R² evaluation
5
Training Configuration

LR: 5e-4, weight decay: 0.01, warmup: 100 steps, early stopping patience: 100.

Best checkpoint selected by validation MAE

Key Observations

MethylGPT Outperforms BMFM

MethylGPT achieves 9% better MAE (4.39 vs 4.79 years) with 12× fewer parameters (~2M vs ~25M). This suggests simpler architectures may be more efficient for methylation age prediction.

Feature Subset Impact

BMFM-RNA used only 2,048 of 8,000 CpG sites. Using all 8K CpGs may improve performance by training all CpG embeddings properly instead of a fixed subset.

Pretraining Success

MLM pretraining achieved excellent reconstruction (PCC=0.997), indicating the encoder learns meaningful methylation patterns. The gap may be in the fine-tuning stage.

Both Models Consistent

Both models show excellent reproducibility: MethylGPT ±0.17 MAE, BMFM-RNA ±0.14 MAE. Results are reliable across different random seeds.

Next Steps

1. Train with Full 8K CpGs

Run BMFM pretraining and fine-tuning using all 8,000 CpG sites instead of 2,048 subset. This ensures all CpG embeddings are properly trained.

2. Multi-Seed Evaluation

Run BMFM with multiple seeds (40-44) to enable fair comparison with MethylGPT's 5-seed evaluation and assess variance.

3. Architecture Optimization

Explore smaller BMFM configurations or different pooling strategies to improve efficiency while maintaining performance.
