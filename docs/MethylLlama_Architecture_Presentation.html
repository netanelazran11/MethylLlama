Why 19,608 CpGs?
The pretrain tokenizer was built on 49,156 CpG sites. AltumAge measures 21,368 of these, but 1,760 are NOT in the pretrain vocabulary. We use all 19,608 that are in both — no feature selection needed, no data leakage.
Why Not the Original 8,000 CpG Subset?
The 8k CpGs were selected by age correlation across the entire AltumAge dataset including the test set → data leakage. Using all 19,608 in-vocab CpGs eliminates this bias entirely.

Section 4

Architecture Components

4.1 RMSNorm

BERT — LayerNorm
\(\displaystyle \text{LN}(x) = \frac{x - \mu}{\sigma} \cdot \gamma + \beta\)
  • Computes mean AND variance
  • Two passes over the vector
  • Heavier compute overhead
MethylLlama — RMSNorm
\(\displaystyle \text{RMSNorm}(x) = \frac{x}{\text{RMS}(x)} \cdot \gamma\)

\(\displaystyle \text{RMS}(x) = \sqrt{\frac{1}{d}\sum_i x_i^2}\)
  • No mean subtraction
  • ~40% faster in practice
  • Used in LLaMA, Mistral, Gemma
  • float32 → cast back to fp16

4.2 Pre-LN vs Post-LN

BERT — Post-LN (normalize AFTER residual)
h = LayerNorm(h + Attn(h))
h = LayerNorm(h + MLP(h))
MethylLlama — Pre-LN (normalize BEFORE sublayer)
h = h + Attn(RMSNorm(h))
h = h + MLP(RMSNorm(h))
PropertyPost-LN (BERT)Pre-LN (LLaMA)
Gradient flow at step 0Can explode / vanishStable from the start
LR warmup requirementMandatory, sensitiveLess critical
Training stabilityNeeds careful tuningStable default
Convergence speedSlower (needs warmup)Faster
For a model training on 24k-token sequences with 49k-site reconstruction, training instability is a real risk. Pre-LN eliminates this from step 1.

4.3 SwiGLU Feed-Forward Network

BERT FFN — GELU (2 matrices)
\(\text{FFN}(x) = W_2 \cdot \text{GELU}(W_1 x)\)

Params per layer: 2 × D × 4D = 8D²

MethylLlama — SwiGLU (3 matrices)
\(\text{FFN}(x) = W_{\text{down}} \big( \text{SiLU}(W_{\text{gate}} x) \odot W_{\text{up}} x \big)\)

Params per layer: 3 × D × (2/3 × 4D) = 8D²  (same budget!)

Why SwiGLU:

  • Gated mechanism: each dimension has a learned gate controlling information flow
  • SiLU = x × sigmoid(x) — smooth, avoids dying-ReLU
  • Consistently outperforms GELU-FFN (PaLM, LLaMA, Gemma)
  • intermediate_size = round(2/3 × 4 × 768 / 64) × 64 = 2048

4.4 Rotary Position Embedding (RoPE)

BERT — Absolute Position Table
embedding_table = nn.Embedding(
    max_seq_len, hidden_size)
# 8002 × 768 ≈ 6.1M parameters
# Fails on unseen lengths
MethylLlama — RoPE
# Zero learned parameters
inv_freq[k] = 1 / (θ ^ (2k / head_dim))
# Rotate Q and K by position
q_rot = q*cos + rotate_half(q)*sin
PropertyAbsolute (BERT)RoPE
Learned parameters~6.1M for L=24,5800
Unseen sequence lengthsFails (untrained positions)Generalizes
Position signal typeAdditive, globalRelative, per-head
MemoryExtra embedding tableCached cos/sin only
Each attention head independently learns how position-sensitive to be — some heads may attend locally (nearby CpGs in a gene), others globally (distant co-methylation patterns).

4.5 ScaleAdaptEncoder (Beta Value Encoding)

Problem with 2-layer MLP
Beta values are continuous in [0, 1]. A 2-layer MLP may not resolve fine differences like 0.85 vs 0.87, which are biologically meaningful for age prediction.

Solution — Sinusoidal basis encoding:

# For each beta value β:
features_k = [sin(β × f_k), cos(β × f_k)]   for k = 1..48
output = Linear(96 → 768)(concat(all features))

# Frequencies f_k ~ N(0, 2π × basis_scale) are TRAINABLE
# The model learns which resolutions of beta variation matter biologically

Special token routing:

β < 0   →  learned nn.Embedding   (MASK→0, CLS→1, SEP→2, PAD→3)
β ≥ 0   →  sinusoidal basis       (β=0 is real: fully unmethylated CpG)
Property2-layer MLP (SCBert)ScaleAdaptEncoder
Resolution of [0,1]Smooth global mapping onlyMultiple frequency bands
Frequency adaptationFixed architectureTrainable frequencies
Special token handlingSeparate logicNative routing (β < 0)
IntegrationMonkey-patched post-initBuilt into __init__
basis_scale1.5 (scRNA, range 0–10+)2.0 (methylation [0,1])

4.6 cpg_scale Initialization

# Embedding fusion formula
h = cpg_scale × CpgEmbed(cpg_ids) + ScaleAdaptEncoder(beta_values)

# cpg_scale = nn.Parameter(torch.tensor(0.1))  ← single learned scalar
Why initialize at 0.1?
At step 0, the beta encoder dominates → the model focuses on methylation levels first. CpG identity (which site) is slowly learned as the model discovers which sites matter most. A healthy training run shows cpg_scale growing from 0.1 toward ~0.3–0.5.
Failure Mode if cpg_scale is Large at Init
If initialized at 1.0, the model collapses to predicting per-site averages (mean beta per CpG across the training set) — ignoring individual sample variation entirely. This gives low MSE but zero biological information.
Section 5

WCED Pretraining Objective

What is WCED?

Whole-Context Encoder Distillation — the model sees 50% of CpGs and must reconstruct ALL CpGs from the CLS token alone.

Full methylation profile: [c₁, c₂, c₃, ..., c₄₉₁₅₆]
↓   random 50% split
View (input to encoder): [CLS, c₃, c₇, c₁₂, ...] ← 24,579 tokens seen
↓   MethylLlamaModel
CLS embedding: [B, 768] ← must encode entire methylation profile
↓   WCEDDecoder
Predictions: [ĉ₁, ĉ₂, ..., ĉ₄₉₁₅₆] ← all 49,156 sites
↓   loss
MSE only on NON-input sites ← must generalize beyond input

Loss Function

\(\mathcal{L} = \mathcal{L}_{\text{recon}} + \lambda_{\text{age}} \cdot \mathcal{L}_{\text{age}}\)
TermFormulaPurpose
ReconstructionMSE(predicted, true) on non-input, non-NaN CpGsCompress full profile into CLS
Age MSEMSE(age_head(CLS), chronological_age)Prevent CLS collapse; encode age
InfoNCESymmetric cross-entropy on similarity matrixDisabled in first run (λ=0)
Why Age Auxiliary Loss?
Without it, the CLS token can collapse to encoding mean methylation level (easy shortcut — MSE drops fast but no biological signal is learned). The age prediction head forces CLS to capture chronological age signal even during unsupervised pretraining.

Metrics Logged

Why Larger Pretraining Corpus Matters

169,000 × 49,156 CpGs vs typical clock models that train on 500–5,000 samples:

  1. Biological variation coverage: 169k samples span diverse tissues, ages, conditions — the model learns which CpG patterns are universal vs tissue-specific
  2. CpG co-methylation patterns: With 49k CpGs per sample, the encoder learns long-range correlations between distant sites (e.g., methylation at one gene promoter predicts another)
  3. CLS quality: A CLS trained to reconstruct all 49k CpGs from 24.5k inputs must compress far more information than a model trained on 8k CpGs → richer representations for downstream tasks
  4. Transfer to 19,608 CpGs: All 19,608 fine-tune CpGs were seen during pretraining — the encoder already knows these sites' methylation dynamics from 169k samples
  5. Age signal without labels: The age auxiliary head during pretraining means CLS already encodes chronological age before any labeled age data is shown in fine-tuning
