What is the CLS token?
After the transformer processes all 8,000 CpG tokens, there is a special token called CLS (Classification token). Its job is to be a single summary of the entire methylation profile. All fine-tuning methods read from this one 512-dim vector to predict age.

MLM vs WCED — key difference:
MLM uses all 8,000 CpGs every batch but masks 25%. The model only predicts masked positions. WCED uses only 4,000 random CpGs per view and predicts all 8,000. MLM gives denser coverage; WCED forces a global summary via the CLS representation — which is why WCED's CLS already encodes age (R²≈0.90) before fine-tuning, while MLM's mean pool must learn age from scratch.

Strategy B — Phase 1
WCED Pretraining
Before predicting age, we train the encoder to understand methylation biology. WCED forces the CLS token to encode the full methylation profile of each person.
How WCED works:
For each person, we take two different random subsets of 4,000 CpGs (out of 8,000). The encoder sees each subset separately, produces a CLS for each — then must reconstruct all 8,000 CpG values from that single CLS vector. This forces CLS to memorize the full profile, not just the CpGs it was given.

Decoder Input
Raw CLS hidden state from encoder output (position 0 of sequence)
The CLS vector is used as-is, straight from the transformer.
pooler_output — CLS goes through an extra Dense(512,512) + Tanh layer first
The pooler is a non-linear projection added by BERT. It gives the decoder a more compressed, stable summary of CLS before reconstruction.
Decoder Architecture
Single linear layer:
Linear(512 → vocab_size)
No activation — raw logits
Works for gene expression because outputs are unbounded counts.
Deeper MLP:
Linear → LayerNorm → GELU → Dropout → Linear → Sigmoid
Sigmoid forces output to 0–1, matching beta value range
Beta values (methylation level) are always between 0 and 1 by definition. Sigmoid guarantees the decoder never predicts invalid values.

IsZeroFocal handles zero-inflation: gene expression data has many exact zeros (gene not expressed), and a standard MSE underweights these errors.

Why We Added Contrastive Learning + Age Supervision
Problem with bare WCED on methylation

In scRNA-seq, gene expression varies dramatically between cell types — CLS naturally learns sample identity.

In methylation, CpG beta values vary much less between people. Without extra supervision, CLS collapses: it learns to predict population-average beta values instead of individual-specific patterns.

How we solved it

InfoNCE (contrastive): Two random subsets of the same person must produce similar CLS. Different people must be different. Forces CLS to encode individual identity.

Age auxiliary head: CLS → age prediction during pretraining directly injects age signal. Result: frozen CLS achieves R²≈0.90 before any fine-tuning.

Key Concept
What Does "Frozen" Mean?
Frozen encoder means the 27 million parameters of the transformer are locked — they do not change during fine-tuning.

Only the small MLP age head (~165k parameters) is trained. The head learns to read the CLS representation that the encoder already produces from pretraining.

Why freeze? The encoder was trained for 190 epochs with 3 objectives. If we let it train on age alone, it forgets everything else — this is called catastrophic forgetting.

Method 3
LoRA Adaptation
R² = 0.873
MAE = 6.64 years

LoRA injects tiny trainable adapter matrices inside the frozen attention layers. The original 27M weights never change — only 98k small adapter parameters (0.4% of the encoder) train.

LoRA math:
Each attention layer computes: output = W_original × input + (α/rank) × B( A(input) )
A and B are small matrices (512×8 and 8×512). B is initialized to zero so the adapter starts with zero effect and learns gradually.
Why it underperforms the frozen baseline: WCED pretraining shaped the attention patterns over 190 epochs with 3 objectives. Even a 0.4% change to query/value projections disrupts these finely-tuned patterns — and the CLS representation degrades. The evidence: when the LoRA adapter was accidentally kept at zero (due to a scheduler bug), the result was 0.920 — better than when LoRA actually trained (0.873). Any modification to the encoder hurts.

Key findings:
(1) WCED Method 1 (frozen encoder + fixed 8k) is the best approach overall (R²=0.9327).
(2) MLM achieves competitive results (mean R²=0.9273, best seed 0.9316) — very close to WCED's best.
(3) Any modification to the WCED encoder — even gentle LoRA — degrades performance below the frozen baseline.
(4) WCED's key advantage: even without fine-tuning, the CLS representation already encodes age (linear probe R²≈0.90), demonstrating richer pretraining signal.

Both MLM and WCED pretraining produce strong age-prediction representations from the same SCBert encoder. WCED's best result (R²=0.9327, MAE=4.61yr) edges out MLM's best seed (R²=0.9316) by a narrow margin. MLM is more reproducible (5-seed mean R²=0.9273 ± 0.004), while WCED's key advantage is that its CLS representation already encodes age before any fine-tuning (linear probe R²≈0.90) — demonstrating richer self-supervised signal. In both strategies, the simplest approach wins: frozen encoder + fixed 8k input + lightweight MLP head.
